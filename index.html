Uploadcare Tutorial

Your Example Page with Form

This is an example demonstrating how to quickly integrate Uploadcare into the file upload form on your website:

  • Sign up at Uploadcare.com
  • Create a project
  • Add UPLOADCARE_PUBLIC_KEY JavaScript variable, with a public key of your project as a value
  • In your form, replace the file input:
    <input type="file">
    with hidden input having "uploadcare-uploader" class:
    <input type="hidden" role="uploadcare-uploader" name="my_file">
